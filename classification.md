# Data Structures Classification Notes

Based on our DSA unit, here is the breakdown of how data is organized:

## 1. Primitive Data Structures
These are the most basic building blocks predefined by the language.
* **Integer:** Whole numbers (e.g., 5, -10).
* **Float/Double:** Decimal numbers.
* **Character:** Single symbols or letters (e.g., 'A').
* **Boolean:** True or False values.

## 2. Non-Primitive Data Structures
More complex structures that group multiple values.

### A. Linear Data Structures (Sequential)
* **Static (Arrays):** Fixed size, stored in contiguous memory. This is what I implemented in C!
* **Dynamic:** Size can change (Linked Lists, Stacks, Queues).

### B. Non-Linear Data Structures (Hierarchical)
* **Trees:** Parent-child relationships (e.g., Binary Trees).
* **Graphs:** Connected nodes (e.g., social media networks).
